# Rhino

This is a small Ruby Web Framework. Rails-like framework.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rhino-framework'
```

And then execute:
```ruby
$ bundle
```

Or install it yourself as:
```ruby
$ gem install rhino-framework
```
